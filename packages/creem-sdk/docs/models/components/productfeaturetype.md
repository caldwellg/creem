# ProductFeatureType

The type of the feature: `custom` (private note), `file` (downloadable files), `licenseKey` (license key), or `customerCredits` (customer credit grant).

## Example Usage

```typescript
import { ProductFeatureType } from "creem/models/components";

let value: ProductFeatureType = "file";
```

## Values

```typescript
"custom" | "file" | "licenseKey" | "customerCredits"
```
